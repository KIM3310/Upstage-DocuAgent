# LMS Import Checklist (SCORM 2004 / IMS CC 1.3)

## 1) 패키지 전 점검
- ZIP 내부에 `imsmanifest.xml` 존재
- `index.html` 경로가 manifest에 정의된 경로와 일치
- 파일명이 특수문자 없이 정상 (공백은 언더스코어 권장)

## 2) 업로드/임포트 단계
- 새 코스(혹은 모듈) 생성
- “콘텐츠 추가/가져오기”에서 SCORM 2004 또는 IMS CC 1.3 선택
- ZIP 업로드 후 항목 생성 확인

## 3) 실행/표시 확인
- 콘텐츠 실행(런치) 시 정상 로드 여부 확인
- 제목/요약/키워드가 메타데이터에 반영되는지 확인
- 학습자 권한으로 접근 테스트

## 4) 검증 로그 체크
- LMS 시스템 로그에서 manifest 파싱 오류 확인
- 경로 에러(404) 발생 여부 확인
- 한글/특수문자 표시 깨짐 여부 확인

## 5) 문제 발생 시 빠른 점검
- `imsmanifest.xml`의 namespace/스키마 위치 오류 여부
- `href` 경로와 ZIP 내부 구조 불일치 여부
- 파일 인코딩(UTF-8) 확인
